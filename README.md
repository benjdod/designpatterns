### Design Patterns

This is a collection of common design patterns used in Java. 

**Note:** If you have opened this folder in Eclipse, this repository will not work correctly. In order to run correctly, you have to open each subfolder as its own project (e.g. open .../designpatterns/iterator/paintsample/ as its own project).

---

#### Factory
- **Bag of Cats (*singleton*):**<br> A single, systemwide bag of cats which we can add and remove cats from. 

#### Iterator
- **Paint Sample (*custom iterator*):**<br> An iterator which iterates over paint samples of a certain color. Exciting.
